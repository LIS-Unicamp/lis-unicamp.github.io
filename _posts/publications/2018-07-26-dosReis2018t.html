Transparência de dados requer a publicação estruturada dos dados para outros sistemas. Dados abertos interconectados que são disponibilizados na Web sem restrições de “copyright” são a chave para atingirmos mecanismos de transparência. Mas muitas vezes os dados a serem publicados pela organização se encontram em diversos sistemas isolados. Neste artigo, propomos um framework para possibilitar a publicação de dados interconectados abertos, partindo de diversas fontes de dados visando a transparência.
